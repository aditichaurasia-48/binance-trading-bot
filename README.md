This project implements a CLI-based Binance Futures Trading Bot.
It supports MARKET and LIMIT orders with validation and logging.
A dry-run mode is used to simulate order execution during submission.

# Binance Futures Trading Bot (CLI)

This project is a CLI-based Binance Futures Trading Bot built using Python.

## Features
- Market and Limit orders
- BUY / SELL support
- Input validation
- Logging support
- Dry-run mode for safe testing
